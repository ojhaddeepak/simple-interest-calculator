# Simple Interest Calculator

This project is a simple interest calculator implemented using a Bash script.

## Description
The script calculates **Simple Interest** based on user-provided inputs:
- Principal amount
- Rate of Interest
- Time period

## Formula Used
Simple Interest = (Principal × Rate × Time) / 100

## Usage
Run the script and provide the required inputs when prompted.

## Example
If Principal = 1000, Rate = 5, Time = 2  
Then Simple Interest = 100
